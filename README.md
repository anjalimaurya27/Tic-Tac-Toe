# Tic Tac Toe Game

A simple Tic Tac Toe game built using HTML, CSS and Javascript.
This project demonstrates basic front-end development and recreates the classic two-player game with a clean interface.

## Features

* Simple and clean user interface
* Two-player classic Tic Tac Toe game
* Built using basic HTML, CSS and Javascript
* Beginner-friendly project

## How to Play

1. Open the project folder.
2. Run the **index.html** file in your browser.
3. Two players take turns marking **X** and **O**.
4. The first player to get three marks in a row wins.

## Technologies Used

* HTML
* CSS
* Javascript

## Project Purpose

This project was created to practice basic front-end development and understand how simple web games are structured.
